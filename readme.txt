Что это за код: онлайн-калькулятор для подсчета прибыльности портфеля, состоящего из акций различных компаний.

Для запуска примера кода необходимо загрузить этот код на какой-либо сервер (подойдет и локальный).

Если этого не сделать и запускать просто как статичный файл, то будет работать не правильно (из-за политик безопасности браузеров - т.к. данный код подгружает внешниие html-файлы/шаблоны, а политика безопасности браузера позволяет JavaScript-у это делать только из одного общего домена).
